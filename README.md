# 《永宁路18号》—— 网页ARG游戏

## 项目介绍

《永宁路18号》是一款**纯网页ARG（替代现实游戏）**。玩家以第一人称视角，通过浏览一个虚构的“安居房产中介”网站，从房源信息、论坛帖子、档案记录、博客日记等碎片化信息中，拼凑出江陵县永宁路18号背后的真相。


## 故事背景

2026年3月，你的大学室友苏晚在回老家江陵县后突然失踪。她的最后一条消息是一张照片——一把老式铜钥匙和一页撕掉的日记，上面写着：

> *“永宁路18号，别信任何人。”*

你唯一能做的，是从她生前浏览过的网站开始，一步步查下去。
